# Ospitly

**Modern hospitality management platform for vacation rental hosts**

[English](#english) | [Italiano](#italiano)

---

## English

### Overview

Ospitly is a modern hospitality management platform designed for vacation rental hosts. Built with React and Vite, it provides a streamlined interface for managing properties, bookings, and guest communications across multiple rental platforms.

### Key Features

- **Property Management** - Centralized dashboard for all your rental properties
- **Booking Overview** - Unified view of reservations across platforms
- **Guest Communication** - Streamlined messaging and check-in instructions
- **Calendar Sync** - Multi-platform calendar integration
- **Analytics Dashboard** - Revenue tracking and occupancy metrics
- **Modern UI** - Clean, responsive interface built with React

### Tech Stack

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

### Quick Start

```bash
# Clone repository
git clone https://github.com/fracabu/ospitly.git
cd ospitly

# Install dependencies
npm install

# Start development server
npm run dev
```

Application available at `http://localhost:5173`

### Commands

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
npm run lint     # Run ESLint
```

## Italiano

### Panoramica

Ospitly e una piattaforma moderna di gestione hospitality progettata per host di affitti vacanze. Costruita con React e Vite, fornisce un'interfaccia snella per gestire proprieta, prenotazioni e comunicazioni con gli ospiti attraverso multiple piattaforme di rental.

### Funzionalita Principali

- **Gestione Proprieta** - Dashboard centralizzata per tutte le tue proprieta in affitto
- **Overview Prenotazioni** - Vista unificata delle prenotazioni tra piattaforme
- **Comunicazione Ospiti** - Messaggistica semplificata e istruzioni check-in
- **Sync Calendario** - Integrazione calendario multi-piattaforma
- **Dashboard Analytics** - Tracking revenue e metriche occupazione
- **UI Moderna** - Interfaccia pulita e responsiva costruita con React

### Stack Tecnologico

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
