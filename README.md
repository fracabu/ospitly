<a name="english"></a>
## :gb: English

### Overview

Ospitly is a modern hospitality management platform designed for vacation rental hosts. It provides a streamlined interface for managing properties, bookings, and guest communications across multiple rental platforms.

### Features

- **Property Management** - Centralized dashboard for all rental properties
- **Booking Overview** - Unified view of reservations across platforms
- **Guest Communication** - Streamlined messaging and check-in instructions
- **Calendar Sync** - Multi-platform calendar integration
- **Analytics Dashboard** - Revenue tracking and occupancy metrics
- **Responsive UI** - Clean interface optimized for all devices

### Quick Start

```bash
git clone https://github.com/fracabu/ospitly.git
cd ospitly
npm install
npm run dev
```

### Commands

```bash
npm run dev      # Development server
npm run build    # Production build
npm run preview  # Preview build
npm run lint     # ESLint
```

<a name="italiano"></a>
## :it: Italiano

### Panoramica

Ospitly e una piattaforma moderna di gestione hospitality progettata per host di affitti vacanze. Fornisce un'interfaccia snella per gestire proprieta, prenotazioni e comunicazioni con gli ospiti su multiple piattaforme.

### Funzionalita

- **Gestione Proprieta** - Dashboard centralizzata per tutte le proprieta
- **Overview Prenotazioni** - Vista unificata prenotazioni cross-platform
- **Comunicazione Ospiti** - Messaggistica e istruzioni check-in semplificate
- **Sync Calendario** - Integrazione calendario multi-piattaforma
- **Dashboard Analytics** - Tracking revenue e metriche occupazione
- **UI Responsiva** - Interfaccia pulita ottimizzata per tutti i dispositivi

### Avvio Rapido

```bash
git clone https://github.com/fracabu/ospitly.git
cd ospitly
npm install
npm run dev
```

### Comandi

```bash
npm run dev      # Server sviluppo
npm run build    # Build produzione
npm run preview  # Preview build
npm run lint     # ESLint
```
